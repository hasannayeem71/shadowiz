# A Shadow Maker

## Installation

```
npm i shadowizi --save
```

### Then...

```
import {shadowiz} from 'shadowizi';

shadowiz({
    shadow_type:'soft',
    padding:false
});
```

## Options

Shadowiz support 2 options, both of which are optional

- _shadow_type_ - _hard | soft_ (Default to soft)
- _padding_ - _boolean_ (Default to false)
